# zoj-solutions-fish_ball


呆滞的慢板的(id:fish_ball, [ranklist 23](http://acm.zju.edu.cn/onlinejudge/showRankList.do?contestId=1&from=0&order=AC))在[zoj](http://acm.zju.edu.cn)的题解,总共有600+题目的代码和解题报告.他的代码十分规范,十分"c++",值得学习.


OJ在提交运行时定义了宏：

`#define ONLINE_JUDGE`

因此，可以用

`#ifndef ONLINE_JUDGE`

`#endif`

来加入调试信息
